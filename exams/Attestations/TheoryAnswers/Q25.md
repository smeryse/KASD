### 25. Merge и split в treap; сложность; применение к последовательностям
`Split` режет дерево на два по ключу/позиции за `O(log n)`; `Merge` склеивает, выбирая корень по приоритету, тоже `O(log n)`. Неявный treap позволяет делать вставки, удаления, развороты отрезков через комбинации split+merge.  
```csharp
(Node?,Node?) Split(Node? v,int key){
    if(v==null) return (null,null);
    if(key<=v.key){ var (l,r)=Split(v.l,key); v.l=r; Update(v); return (l,v); }
    else { var (l,r)=Split(v.r,key); v.r=l; Update(v); return (v,r); }
}
```
