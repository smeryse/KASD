### 20. Разреженная таблица: задачи, ограничения; предобработка/запрос; почему `O(1)`
Sparse table предназначена для статических idempotent-операций (min/max/gcd), где обновлений нет. Предобработка: `st[k][i]` — результат на отрезке длиной `2^k` от i, строится за `O(n log n)`. Запрос `[l,r]` берёт два перекрывающихся блока длины `2^k` и комбинирует за `O(1)`. Обновления невозможны без перестроения.  
```csharp
int Query(int l,int r){ int k=lg[r-l+1]; return Math.Min(st[k,l], st[k,r-(1<<k)+1]); }
```
