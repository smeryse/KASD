### 40. Идеальное хеширование; двухуровневая схема; кукушкин хеш
Идеальное хеширование — отсутствие коллизий на фиксированном наборе: первый уровень распределяет элементы, второй для каждого бакета подбирает хеш без коллизий. Кукушкин хеш: два хеша/две позиции на элемент, при вставке может “выселять” другие по их альтернативным позициям; при цикле — перестройка. Хорошо работает при умеренном заполнении.
