### 44. Статические члены и конструктор; `this`
Статические члены принадлежат типу (один экземпляр на класс), статический конструктор выполняется один раз перед первым использованием. `this` ссылается на текущий объект, позволяет различать поля и параметры и передавать сам объект.  
```csharp
class Sample{ static int Created; public Sample(){Created++;} public void Show()=>Console.WriteLine(this.ToString()); }
```
