### 48. Перегрузка методов; анонимные типы и применение
Перегрузка — несколько методов с одним именем и разными параметрами, компилятор выбирает по сигнатуре. Анонимные типы (`new { Name="x", Age=1 }`) создаются на лету, часто в LINQ для временных проекций; свойства только для чтения.  
```csharp
int Sum(int a,int b)=>a+b; int Sum(int a,int b,int c)=>a+b+c;
```
