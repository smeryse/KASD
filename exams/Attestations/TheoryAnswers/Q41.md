### 41. Фильтр Блума: операции; плюсы/минусы; вероятность ложноположительного
Битовый массив + k хешей. `Add` ставит k бит, `Query` проверяет, что все они 1 — ложноположительные возможны, ложноотрицательные нет (при корректной реализации). Вероятность FP ≈ `(1 - e^{-(k n / m)})^k`; уменьшается при большем m и оптимальном k. Плюсы — компактность и скорость; минусы — нет удаления (кроме счётчиков) и FP.  
```csharp
void Add(string s){ foreach(var h in Hashes(s)) bits[h%M]=true; }
bool MightContain(string s){ return Hashes(s).All(h=>bits[h%M]); }
```
