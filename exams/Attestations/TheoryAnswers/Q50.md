### 50. Делегат; многоадресный; событие vs делегат; анонимный метод; шаблон `EventArgs`
Делегат — типизированная ссылка на метод; многоадресный хранит список методов. Событие — обёртка над делегатом, ограничивающая внешние вызовы (только подписка/отписка). Анонимный метод/лямбда задаёт обработчик inline. Шаблон событий: `EventHandler(object sender, EventArgs e)`, свои данные — наследование от `EventArgs`.  
```csharp
public delegate void Handler(string msg);
public event Handler OnEvent;
public void Raise()=>OnEvent?.Invoke("hi");
```
