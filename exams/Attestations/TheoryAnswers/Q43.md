### 43. Конструктор; модификаторы доступа; инкапсуляция; свойства (в т.ч. авто)
Конструктор инициализирует объект. Модификаторы (public/private/protected/internal) задают видимость. Инкапсуляция — скрываем данные, открываем управляемый интерфейс (свойства/методы) для поддержания инвариантов. Свойства могут содержать логику get/set или быть авто (компилятор создаёт поле).  
```csharp
class Counter{ private int val; public int Value=>val; public void Inc()=>val++; }
```
