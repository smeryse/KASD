### 46. Когда вызывать конструктор базового класса; `base`
Если базовый требует параметров или инициализации, производный вызывает его через список инициализации `: base(args)`. `base.Member` используют, чтобы явно обратиться к базовой реализации или конструктору.  
```csharp
class B{ public B(int x){} }
class D:B{ public D():base(42){} }
```
