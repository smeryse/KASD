### 34. Компоненты сильной связности; алгоритм Косарайю; конденсация
КСС — максимальные подграфы, где вершины достижимы друг из друга. Косарайю: 1) DFS на исходном графе, записываем вершины по времени выхода; 2) DFS на транспонированном графе в обратном порядке списка — каждая волна даёт компоненту. Конденсация — DAG компонент, где дуги отражают связи между ними.  
```csharp
void Dfs1(int v){ used[v]=true; foreach(var to in g[v]) if(!used[to]) Dfs1(to); order.Add(v); }
void Dfs2(int v,int c){ comp[v]=c; foreach(var to in gr[v]) if(comp[to]==-1) Dfs2(to,c); }
```
