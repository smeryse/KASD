### 51. Исключения: зачем; `try-catch-finally`; когда выполняется `finally`; `throw`; свои исключения
*(Вопрос отсутствует в лекционных конспектах; краткий ответ приведён для полноты.)*  
Исключения сигнализируют об ошибках. `try-catch-finally` перехватывает и даёт гарантированный блок `finally`; `finally` не выполняется только при аварийном завершении процесса. `throw` генерирует исключение; собственные исключения наследуют от `Exception`, добавляя контекст.  
```csharp
try{ Do(); } catch(Exception ex){ Console.WriteLine(ex.Message); } finally{ Clean(); }
```
