### 47. Полиморфизм; `virtual`; `override` vs `new`; абстрактный класс/метод; экземпляр абстрактного класса
Полиморфизм — единый интерфейс, разные реализации. `virtual` разрешает переопределение. `override` заменяет базовую реализацию с полиморфизмом; `new` скрывает, но при обращении через базовый тип сработает базовый метод. Абстрактный класс/метод задаёт обязательный контракт; абстрактный класс нельзя инстанцировать.  
```csharp
abstract class Shape{ public abstract double Area(); }
class Square:Shape{ public double A; public override double Area()=>A*A; }
```
