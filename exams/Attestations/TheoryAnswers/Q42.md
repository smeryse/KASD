### 42. Класс и объект; связь; элементы класса
Класс описывает поля/свойства/методы/события/конструкторы — абстракция сущности. Объект — экземпляр класса с конкретным состоянием. Связь: класс — шаблон, объект — конкретизация; через класс создаём множество однотипных объектов.  
```csharp
class Point{ public int X{get;set;} public int Y{get;set;} public Point(int x,int y){X=x;Y=y;} }
```
