### 31. Центроидная декомпозиция: центроид; построение; сравнение с HLD
Вопрос из pdf: “Что такое центроидная декомпозиция? Как находится центроид? Опишите алгоритм построения и сравните с HLD.”  
Центроид — вершина, при удалении которой каждая компонента имеет размер ≤ n/2 (n — размер текущего дерева/компоненты). Нахождение: DFS для размеров поддеревьев, затем спускаемся в ребёнка, если его поддерево > n/2, пока не найдём подходящую вершину.  
Построение: 1) ищем центроид компоненты, добавляем его в дерево центроидов; 2) “удаляем” его, рекурсивно строим декомпозицию для получившихся поддеревьев; 3) получаем дерево центроидов (родитель — центроид верхнего уровня). Используется для задач расстояний/пар с разложением по слоям центроидов.  
Сравнение с HLD: HLD режет дерево на тяжёлые пути и отвечает запросы на путях (u–v) через `O(log n)` отрезков; центроидная декомпозиция режет по центрам, удобна для задач, где важно расстояние/покрытие от набора вершин. Структура — дерево центроидов, а не интервальная раскладка пути.
