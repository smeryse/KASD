### 45. Наследование; синтаксис; какие члены наследуются; sealed; ограничения
`class Child : Parent` — наследует открытые/защищённые члены (учитывая модификаторы), может переопределять виртуальные. Запечатанный (`sealed`) класс нельзя расширять. В C# — одиночное наследование классов.  
```csharp
class Animal{ public virtual void Sound(){} }
class Cat:Animal{ public override void Sound()=>Console.WriteLine("Meow"); }
```
