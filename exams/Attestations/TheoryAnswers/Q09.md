### 9. Рандомизированный QuickSelect. Средняя сложность. Медиана медиан. Плохой фиксированный pivot
QuickSelect: выбираем pivot (случайный), partition, идём только в одну часть. Средняя сложность `O(n)` — математическое ожидание на случайных опорах; худшее `O(n^2)` при систематически плохих pivot (например, всегда первый элемент на отсортированном массиве). Алгоритм “медиана медиан” выбирает хороший pivot детерминированно и гарантирует `O(n)` в худшем случае.  
```csharp
int QuickSelect(int[] a,int k,Random rng){
    int l=0,r=a.Length-1;
    while(true){
        int p=rng.Next(l,r+1);
        int pos=Partition(a,l,r,p);
        if(pos==k) return a[pos];
        if(k<pos) r=pos-1; else l=pos+1;
    }
}
```
