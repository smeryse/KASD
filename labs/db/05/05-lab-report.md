## 1. Листинги DDL-команд

### 1.1. Команда создания базы данных

```sql
-- Создание базы данных
CREATE DATABASE [database_name]
    WITH ENCODING = 'UTF8'
    LC_COLLATE = 'ru_RU.UTF-8'
    LC_CTYPE = 'ru_RU.UTF-8'
    TEMPLATE = template0;
```

### 1.2. Команды создания таблиц (CREATE TABLE)

```sql
-- Таблица 1: [Название таблицы]
-- Описание: [Назначение таблицы]
CREATE TABLE [table_name] (
    [column_name] [data_type] [CONSTRAINTS],
    -- PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL, CHECK
);

-- Таблица 2: [Название таблицы]
CREATE TABLE [table_name] (
    [column_name] [data_type] [CONSTRAINTS]
);

-- ... остальные таблицы
```

### 1.3. Команды создания ограничений целостности (ALTER TABLE)

```sql
-- Добавление внешнего ключа
ALTER TABLE [table_name]
    ADD CONSTRAINT [constraint_name]
    FOREIGN KEY ([column_name])
    REFERENCES [referenced_table]([referenced_column])
    ON DELETE [CASCADE|SET NULL|RESTRICT]
    ON UPDATE [CASCADE|SET NULL|RESTRICT];

-- Добавление ограничения CHECK
ALTER TABLE [table_name]
    ADD CONSTRAINT [constraint_name]
    CHECK ([condition]);

-- Добавление ограничения UNIQUE
ALTER TABLE [table_name]
    ADD CONSTRAINT [constraint_name]
    UNIQUE ([column_name]);
```
