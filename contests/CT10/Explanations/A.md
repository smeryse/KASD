# Задача A. Паросочетание

## Идея

Классический **алгоритм Куна** для нахождения максимального паросочетания в двудольном графе.

## Алгоритм

```
Для каждой вершины u в левой доле:
  visited = {}
  DFS(u):
    если u в visited → false
    добавить u в visited
    для каждой соседки v:
      если v свободна ИЛИ DFS(match[v]):
        match[v] = u
        return true
    return false
```

## Почему работает

Каждый успешный DFS находит **augmenting path** — путь от свободной вершины левой доли до свободной вершины правой доли. «Переворот» рёбер на этом пути увеличивает паросочетание на 1.

## Восстановление ответа

Массив `matchB[v] = u` означает, что вершина $v$ из правой доли сопоставлена с $u$ из левой. Проходим по `matchB` и выводим все пары.
