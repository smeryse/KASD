# D. Групповой турнир

## Условие
Распределить результаты нерешённых матчей в круговом турнире так, чтобы каждая команда набрала нужное количество очков.

## Алгоритм
**Максимальный поток**.

### Модель сети
- **Источник S** → каждый нерешённый матч: capacity = 3 (очка за матч)
- **Матч (i,j)** → команда i: capacity = 3
- **Матч (i,j)** → команда j: capacity = 3
- Каждая команда → **сток T**: capacity = remaining (сколько очков ещё нужно)

### Восстановление ответа
Поток из узла матча в команду показывает, сколько очков она получит:
- 3 и 0 → W/L (победа в основное время)
- 0 и 3 → L/W
- 2 и 1 → w/l (победа в овертайме)
- 1 и 2 → l/w

### Сложность
O(V * E²), где V = M + N + 2, M — количество нерешённых матчей.
