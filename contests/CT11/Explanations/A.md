# A. Просто поток

## Условие
Найти максимальный поток в сети с неориентированными рёбрами и вывести поток по каждому ребру.

## Алгоритм
**Эдмондс-Карп** — BFS для поиска увеличивающих путей в остаточном графе.

### Модель
- Неориентированное ребро (u, v) с пропускной способностью c моделируется как:
  - Прямое ребро u→v с capacity c
  - Обратное ребро v→u с capacity c (та же пропускная способность!)
- Источник: вершина 1, сток: вершина N

### Восстановление потоков
После нахождения максимального потока значение flow на каждом оригинальном ребре — это поток через соответствующую трубу.

### Сложность
O(V * E²) — достаточно для N ≤ 100, M ≤ 5000.
