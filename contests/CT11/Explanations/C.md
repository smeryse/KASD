# C. Улиточки

## Условие
Найти два непересекающихся по рёбрам пути из s в t в ориентированном графе.

## Алгоритм
**Максимальный поток с единичными пропускными способностями**.

### Модель
- Каждое ребро имеет capacity = 1
- Находим максимальный поток из s в t
- Если поток ≥ 2, существует два непересекающихся пути
- Восстанавливаем пути DFS по рёбрам с flow = 1

### Почему работает
Каждый единичный поток соответствует одному пути. Два единицы потока = два непересекающихся пути (по рёбрам).

### Сложность
O(E * f*), где f* ≤ 2 — значение потока. Практически O(E).
