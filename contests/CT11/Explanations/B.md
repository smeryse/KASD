# B. Разрез

## Условие
Найти минимальный разрез между вершинами 1 и n в неориентированном графе. Вывести количество рёбер в разрезе, их суммарную пропускную способность и номера рёбер.

## Алгоритм
**Теорема Форда-Фалкерсона**: величина максимального потока равна пропускной способности минимального разреза.

### Шаги
1. Находим максимальный поток алгоритмом Эдмондс-Карп
2. BFS в остаточном графе из source для нахождения достижимых вершин
3. Рёбра минимального разреза — те, у которых одна вершина достижима, другая нет
4. Сортируем номера рёбер и выводим

### Сложность
O(V * E²) для Эдмондс-Карп + O(E) для нахождения разреза.
