# Центроидная декомпозиция дерева

## Условие задачи
Дано дерево. Требуется построить центроидную декомпозицию и ответить на запросы о расстояниях между вершинами.

## Алгоритм
**Центроидная декомпозиция** — рекурсивное разбиение дерева путём удаления центроидов.

### Идея алгоритма
**Центроид** — вершина, удаление которой разбивает дерево на компоненты размером не более n/2.

**Свойства**:
- В любом дереве есть хотя бы один центроид (может быть два)
- Глубина декомпозиции: O(log n)
- Любая вершина принадлежит O(log n) центроидам на разных уровнях

**Построение**:
1. Найти размер каждого поддерева
2. Найти центроид: вершина, где все поддеревья ≤ n/2
3. Рекурсивно обработать компоненты после удаления центроида

**Запрос расстояния**:
Расстояние в дереве = depth[u] + depth[v] - 2 × depth[LCA(u, v)]

### Структуры данных
- `adj[]` — списки смежности
- `removed[]` — удалённые вершины (центроиды)
- `subtreeSize[]` — размер поддерева
- `centroidParent[]` — родитель в дереве центроидов
- `depth[]`, `parent[]` — для вычисления расстояния

### Ход алгоритма
**Построение декомпозиции**:
1. `CalcSubtreeSize(u, parent)`:
   - `subtreeSize[u] = 1`
   - Для каждого ребёнка v: рекурсивно, `subtreeSize[u] += subtreeSize[v]`

2. `FindCentroid(u, parent, total)`:
   - Ищем вершину, где все поддеревья ≤ total/2
   - Если такое поддерево есть → спускаемся в него

3. `BuildCentroidDecomposition(u, parent)`:
   - Найти центроид для текущего дерева
   - Пометить как удалённый
   - Рекурсивно обработать компоненты

**Запрос расстояния**:
1. DFS от u для вычисления depth и parent
2. LCA(u, v) подъёмом по родителям
3. `distance = depth[u] + depth[v] - 2 × depth[lca]`

### Пример работы
```
Вход:
5 2
1-2, 1-3, 2-4, 2-5
Запросы: (1, 5), (3, 4)

Дерево:
      1
     / \
    2   3
   / \
  4   5

Центроидная декомпозиция:
1. Размер дерева = 5
   subtreeSize: [3, 2, 1, 1, 1]
   Центроид: вершина 2 (все поддеревья ≤ 2)

2. После удаления 2:
   Компонента {1, 3}: центроид = 1
   Компонента {4}: центроид = 4
   Компонента {5}: центроид = 5

Дерево центроидов:
      2
     /|\
    1 4 5
    |
    3

Запрос (1, 5):
LCA(1, 5) в исходном дереве = 1
distance = depth[1] + depth[5] - 2×depth[1] = 0 + 2 - 0 = 2

Запрос (3, 4):
LCA(3, 4) = 1
distance = depth[3] + depth[4] - 2×depth[1] = 1 + 2 - 0 = 3

Ответ:
2
3
```

### Сложность
- **Построение**: O(n log n) — каждый уровень обрабатывает все вершины
- **Запрос расстояния**: O(n) в данной реализации (DFS + LCA)
- **Память**: O(n) — граф и массивы

### Ключевые моменты реализации
1. **Удаление вершин**: помечаем центроиды как removed, не удаляем физически
2. **Поиск центроида**: спускаемся в поддерево размером > n/2
3. **Рекурсия**: обрабатываем каждую компоненту связности после удаления
4. **Расстояние**: в данной реализации используется простой DFS + LCA
