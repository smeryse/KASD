# F. Конденсация графа

## Введение

Задача на построение **конденсации ориентированного графа**. Конденсация — это граф, вершинами которого являются компоненты сильной связности (КСВ) исходного графа, а рёбра соответствуют связям между компонентами.

Решение использует **алгоритм Косарайю**, который находит КСВ за два прохода DFS.

## Идея

Алгоритм Косарайю основан на свойстве: если отсортировать вершины по времени завершения DFS в исходном графе, а затем обработать их в обратном порядке в транспонированном графе, то каждый обход DFS выделит одну компоненту сильной связности.

Два этапа:
1. **Прямой DFS** в исходном графе — запоминание порядка завершения
2. **Обратный DFS** в транспонированном графе — выделение компонент

## Разбор кода

### Инициализация

```csharp
adj = new List<int>[n + 1];
adjRev = new List<int>[n + 1];
for (int i = 1; i <= n; i++)
{
    adj[i] = new List<int>();
    adjRev[i] = new List<int>();
}
```

Два списка смежности:
- `adj` — для исходного графа
- `adjRev` — для транспонированного (обратного) графа

```csharp
for (int i = 0; i < m; i++)
{
    line = Console.ReadLine();
    parts = line.Split(' ', StringSplitOptions.RemoveEmptyEntries);
    int u = int.Parse(parts[0]);
    int v = int.Parse(parts[1]);
    if (u == v) continue;  // Игнорируем петли
    adj[u].Add(v);
    adjRev[v].Add(u);  // Обратное ребро
}
```

Петли игнорируются, так как они не влияют на компоненты сильной связности.

### Фаза 1: Прямой DFS

```csharp
visited = new bool[n + 1];
order = new List<int>();

for (int i = 1; i <= n; i++)
{
    if (!visited[i])
        DFS1(i);
}
```

Обход всех вершин для построения порядка завершения.

```csharp
private static void DFS1(int u)
{
    visited[u] = true;
    foreach (int v in adj[u])
    {
        if (!visited[v])
            DFS1(v);
    }
    order.Add(u);  // Добавляем ПОСЛЕ обработки потомков
}
```

Вершина добавляется в `order` после завершения обработки всех её потомков. Это даёт топологический порядок завершения.

### Фаза 2: Обратный DFS

```csharp
component = new int[n + 1];
int componentCount = 0;

order.Reverse();  // Обрабатываем в обратном порядке
foreach (int v in order)
{
    if (component[v] == 0)
    {
        componentCount++;
        DFS2(v, componentCount);
    }
}
```

Обратный порядок гарантирует, что мы начинаем с «стоков» в конденсации. Каждый вызов `DFS2` выделяет одну КСВ.

```csharp
private static void DFS2(int u, int comp)
{
    component[u] = comp;
    foreach (int v in adjRev[u])
    {
        if (component[v] == 0)
            DFS2(v, comp);
    }
}
```

Обход в обратном графе: все достижимые вершины принадлежат той же компоненте сильной связности.

### Подсчёт рёбер конденсации

```csharp
var condensationEdges = new HashSet<(int, int)>();
for (int u = 1; u <= n; u++)
{
    foreach (int v in adj[u])
    {
        if (component[u] != component[v])
            condensationEdges.Add((component[u], component[v]));
    }
}
```

Проходим по всем рёбрам исходного графа. Если концы ребра принадлежат разным компонентам, добавляем ребро в конденсацию. `HashSet` автоматически удаляет кратные рёбра.

### Вывод результата

```csharp
Console.WriteLine(condensationEdges.Count);
```

Выводим количество уникальных рёбер в конденсации.

## Оптимизации

- **Двухпроходный DFS**: линейное время O(n + m)
- **HashSet для рёбер**: автоматическое удаление кратных рёбер
- **Игнорирование петель**: уменьшение размера графа
- **Временная сложность**: O(n + m + k), где k — количество рёбер конденсации
- **Пространственная сложность**: O(n + m)

## Резюме

Алгоритм Косарайю эффективно находит компоненты сильной связности за два прохода DFS. Первый проход строит порядок завершения, второй — выделяет компоненты в обратном графе. Конденсация строится путём проверки рёбер между разными компонентами, с автоматическим удалением кратных рёбер через HashSet.
