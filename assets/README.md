# Algorithms and Data Structures

Учебный репозиторий на C#/.NET без лишней магии. Основная рабочая структура:

- `contests/` - контесты по алгоритмам
- `tasks/` - отдельные учебные задачи
- `labs/` - лабораторные
- `docs/` - лекции и материалы

## Как ориентироваться

Если нужна задача из контеста, открывай папку вида `contests/CT9/`.

Если нужна отдельная учебная задача, открывай `tasks/<номер-название>/`.

Если нужна лабораторная, открывай `labs/`.

## Запуск

Требуется `.NET 8 SDK`.

Примеры запуска:

```bash
dotnet run --project contests/CT9/CT9.csproj -- A sample
dotnet run --project tasks/01-vector-norm/VectorNorm.csproj
```

Проверка по эталонному выводу для контестов:

```bash
dotnet run --project contests/CT9/CT9.csproj -- A sample check
```

## Правило проекта

Код живет в корневых папках `contests/`, `tasks/` и `labs/`. Если появляется альтернативная параллельная структура, ее лучше не поддерживать.
