## Build & Run

### Build a specific contest

```bash
# Navigate to contest folder and build
cd CT1 && dotnet build
cd CT2 && dotnet build
```

### Run a specific contest

```bash
# Navigate to contest folder and run
cd CT1 && dotnet run
cd CT2 && dotnet run
```

## Requirements

- .NET 8.0 SDK or higher
- C# 12
- Visual Studio Code / Visual Studio 2022
